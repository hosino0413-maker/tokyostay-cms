# 待完成项目：分销系统

状态：待完成，后续规划，不在当前版本实现。

目标：
- 管理员可以主动创建用户，并分配 owner / admin / agent / viewer 等权限。
- 分销员填写姓名和联络 ID 后，可以生成自己的专属房源展示链接。
- 客户通过分销链接进入房源页并提交咨询后，线索自动归属到对应分销员。
- 管理员可以查看全部分销员、全部访问、全部咨询和成交状态。
- 分销员只能查看自己的专属链接和自己的客户线索，分销员之间数据完全隔离。

后续需要设计：
- 用户登录与后台密码保护。
- 用户表：姓名、账号、初始密码、角色、联系方式、分销代码、状态。
- 线索表：客户姓名、联系方式、感兴趣房源、来源 refCode、所属 agentId、咨询时间、跟进状态。
- 管理员用户管理页面。
- 分销员专属链接生成页面。
- 客户咨询表单和来源追踪逻辑。
